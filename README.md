# Coding Task



## Requirements

* [git](https://git-scm.com/)
* [vagrant](https://www.vagrantup.com/)
* [ansible](https://www.ansible.com/)


## How to run?

```bash
    git clone
    cd 
    vagrant up

```
Wait until it finish 
and navigate to [localhost:8000](http://localhost:8000)
